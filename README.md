# The Lost Temple - Adventure game

## Author: Mark Borysenko
## Date: 26.03.2026
## Version: 1.0


# Description
This is a text-based adventure game made totally in PHP.
The player explores a jungle and makes choices that affect the outcome of the story.

# Plot
- Multiple story paths (Jungle, River, Hill)
- Different endings:
  - WIN
  - GAME OVER
  - GOOD END
- ASCII art (I took it from internet an AI)
- Input validation (only valid choices allowed)
- Replay option after finishing the game

# How it works
The player chooses between different options (1,2 or 3).
Each choice leads to a different scenario and ending.

# How to run
1. Open terminal
2. Navigate to the project folder
3. Run: 
```cmd
php lost_temple_game.php
```

# Replay
- After the game you can press ENTER to play again
- Type "q" to quit

# Technologies used
- PHP
- CLI (Command Line Interface)
